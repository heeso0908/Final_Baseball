### 1. 확률 기반 모델
        Pythagorean Expectation (피타고리안 기대승률)
        → 득점/실점으로 기대 승률 계산
        → Bill James가 개발
        → 공식: R² / (R² + RA²)
        → 가장 단순하고 널리 쓰임

---


### 2. 몬테카를로 시뮬레이션
        → 경기를 수만 번 반복 시뮬레이션
        → 각 타석을 확률적으로 처리
        → 시즌 승률, 포스트시즌 확률 계산에 사용
        → FanGraphs playoff odds가 이 방식

---

### 3. Markov Chain 모델
        → 24가지 베이스-아웃 상황을 상태로 정의
        → 각 상태 전이 확률로 득점 기대값 계산
        → RE24 지표의 이론적 기반
        → 이닝별 득점 분포 시뮬레이션에 강함

---

### 4. PECOTA / ZiPS / Steamer (선수 성적 예측)
        PECOTA  → Baseball Prospectus 개발
                유사 선수 비교 기반 예측
        ZiPS    → Dan Szymborski 개발
                회귀 분석 + 노화 커브 반영
        Steamer → 가중 평균 기반
                FanGraphs에서 주로 사용

---

### 5. WAR 기반 팀 승률 모델
        → 선수별 WAR 합산 → 팀 승률 추정
        → 공식: 팀 승률 = 0.500 + (WAR합계 / 경기수)
        → 로스터 구성의 이론적 가치 평가

---

### 6. ELO 레이팅
        → 체스에서 유래한 상대적 강도 평가
        → 경기 결과로 레이팅 업데이트
        → FiveThirtyEight MLB ELO가 대표적
        → 홈/원정 보정, 선발투수 보정 포함

---

### 7. 머신러닝 기반
        → XGBoost, Random Forest 등
        → 대량의 Statcast 데이터 학습
        → 타구 결과 예측 (xBA, xSLG 등이 이 방식)
        → 최근 구단 프런트에서 내부적으로 많이 사용

---

분석 목적별:
시즌 승률 예측  → 피타고리안 + 몬테카를로
이닝 득점 분석  → Markov Chain
선수 성적 예측  → ZiPS / Steamer
팀 전력 평가    → WAR 기반
타구 결과 예측  → ML 기반 (Statcast)
